# Вопросы и ответы по поступлению в МАИ

| Ключевые слова | Вопрос | Ответ |
|----------------|--------|-------|
| БВО, базовое высшее, бакалавриат, специалитет, требования, образование | **Какие требования для поступления на программы базового высшего образования (бакалавриат, специалитет)?** | Необходимо иметь среднее общее образование, среднее профессиональное образование или высшее образование, подтвержденное соответствующим документом (аттестат, диплом). |
| магистратура, специализированное высшее, требования, диплом | **Какие требования для поступления на программы специализированного высшего образования (магистратура)?** | Требуется наличие высшего образования, подтвержденного дипломом. |
| документы, аттестат, диплом, иностранный документ, поступление | **Какие документы нужно подать для поступления?** | Нужно предоставить документ об образовании (аттестат о среднем общем образовании, диплом о среднем профессиональном или высшем образовании). Для иностранных документов требуется их признание в РФ. Также нужен документ, удостоверяющий личность. |
| вступительные испытания, ЕГЭ, внутренние экзамены, бакалавриат, магистратура | **Какие вступительные испытания нужно сдавать?** | Для программ базового высшего образования — ЕГЭ или внутренние испытания (для отдельных категорий). Для программ специализированного высшего образования — только внутренние вступительные испытания, проводимые МАИ. |
| предметы, ЕГЭ, профильные испытания, русский язык, математика | **По каким предметам проводятся вступительные испытания для бакалавриата и специалитета?** | Устанавливается МАИ на основе федерального перечня. Обычно это русский язык и два профильных предмета (например, математика и физика для технических направлений). |
| внутренние испытания, ЕГЭ, инвалиды, иностранцы, профессиональное образование | **Кто может сдавать внутренние вступительные испытания вместо ЕГЭ?** | Инвалиды, иностранные граждане, лица с профессиональным образованием, а также те, кто получил среднее образование за рубежом и не сдавал ЕГЭ в текущем году. |
| внутренние испытания, дистанционные технологии, очный формат, русский язык | **Как проводятся внутренние вступительные испытания?** | Очно или с использованием дистанционных технологий (с идентификацией поступающих). Форма — письменная или устная, на русском или иностранном языке (по решению МАИ). |
| минимальные баллы, ЕГЭ, внутренние испытания, порог | **Каковы минимальные баллы для вступительных испытаний?** | Минимальные баллы устанавливаются МАИ и публикуются на сайте. Для внутренних испытаний они соответствуют минимальным баллам ЕГЭ, установленным учредителем. |
| специальности, направления, конкурс, вузы, подача заявлений | **Можно ли подать документы на несколько специальностей?** | Да, можно участвовать в конкурсе по 5 специальностям или направлениям подготовки в МАИ и подать заявления в 5 вузов одновременно. |
| индивидуальные достижения, олимпиады, ГТО, волонтерство, военная служба | **Какие индивидуальные достижения учитываются?** | Документы об образовании с отличием, участие в олимпиадах, спортивные достижения (например, знак ГТО), военная служба, волонтерская деятельность и др. Перечень устанавливает МАИ. |
| баллы, индивидуальные достижения, целевая квота, конкурсные баллы | **Сколько баллов дают за индивидуальные достижения?** | До 10 баллов за общие достижения для бакалавриата и специалитета, плюс 5 баллов за целевые достижения при поступлении на целевую квоту. |
| без экзаменов, олимпиады, Всероссийская олимпиада, международные олимпиады | **Кто может поступить без вступительных испытаний?** | Победители и призеры Всероссийской олимпиады школьников, члены сборных РФ на международных олимпиадах, победители олимпиад из перечня МАИ (по программам бакалавриата и специалитета). |
| квоты, целевая квота, особая квота, отдельная квота, бюджетные места | **Есть ли особые квоты для поступающих?** | Да, есть целевая квота, особая квота (не менее 10% для лиц с ограниченными возможностями здоровья), отдельная квота (не менее 10% для определенных категорий). |
| подача документов, ЕПГУ, почта, личная подача | **Как подать документы?** | Лично, по почте или через Единый портал госуслуг (ЕПГУ). |
| сроки, бюджетные места, подача документов, июнь, июль | **Какие сроки подачи документов на бюджетные места (бакалавриат, специалитет)?** | С 20 июня до 15 июля (12:00) — со вступительными испытаниями, до 25 июля (17:00) — без испытаний. |
| конкурс, баллы, ранжирование, вступительные испытания | **Как проходит отбор поступающих?** | На конкурсной основе по сумме баллов за вступительные испытания и индивидуальные достижения. Ранжирование — по убыванию баллов. |
| зачисление, приказы, август, приоритетный этап, основной этап | **Когда публикуются приказы о зачислении на бюджет (бакалавриат, специалитет)?** | Приоритетный этап: 2–3 августа, основной: 6–7 августа, дополнительный: 10–11 августа. |
| иностранные граждане, квота, платные места, вступительные испытания | **Как иностранные граждане могут поступить в МАИ?** | В рамках квоты на образование иностранных граждан (по направлениям Минобрнауки) или на платные места (по договору). Возможны отдельные вступительные испытания. |
| инвалиды, ограниченные возможности, ассистенты, адаптация, Брайль | **Какая поддержка есть для поступающих с ограниченными возможностями здоровья?** | Увеличение времени на экзамен (до 1 часа), ассистенты, адаптированные условия (шрифт Брайля, звукоусиление и др.) в зависимости от особенностей. |
| апелляция, результаты, вступительные испытания, комиссия | **Можно ли обжаловать результаты вступительных испытаний?** | Да, в день объявления результатов или на следующий рабочий день можно подать апелляцию в апелляционную комиссию МАИ. |
| дополнительный прием, незаполненные места, август, сентябрь | **Проводится ли дополнительный прием?** | Да, если остаются незаполненные места после основного приема. Сроки: с 12 августа (бакалавриат, специалитет) или с 1 сентября (магистратура). |
| расписание, вступительные испытания, сайт МАИ, июнь | **Как узнать расписание вступительных испытаний?** | Расписание публикуется на сайте МАИ не позднее 1 июня 2025 года. |
